🔍 PWA Debug Tools - SweetHR

Check PWA installation requirements and debug issues

📋 Manual Debugging Steps:

  1. Open Chrome DevTools (F12)
  2. Go to Application tab
  3. Check Manifest: Look for errors in manifest.json
  4. Check Service Workers: Verify SW is registered and running
  5. Run Lighthouse: Go to Lighthouse tab → PWA audit
  6. Check Console: Look for any JS errors

🛠️ Common PWA Issues:

  • HTTPS Required: PWA needs HTTPS (or localhost)
  • Missing Icons: Need at least 192x192 and 512x512 icons
  • Invalid Manifest: Check manifest.json syntax
  • Service Worker Errors: Check SW registration
  • Minimum Criteria: Need manifest + service worker + HTTPS
